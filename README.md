# 🧠 Deep Learning - NPTEL Course Repository

This repository contains structured materials for a **12-week NPTEL Deep Learning course**, including:

* 📘 Lecture notes (PDFs)
* 💻 Code implementations
* 📊 Assignments and experiments

---

## 📅 Course Structure (Week-wise)

### 📌 Week 01 – Artificial Neural Networks (ANN)

Introduction to neural networks, perceptron, activation functions, and basic architectures.

### 📌 Week 02 – Optimizers & Regularization

Gradient descent variants, momentum, Adam optimizer, dropout, and regularization techniques.

### 📌 Week 03 – Convolutional Neural Networks (CNN)

CNN fundamentals, convolution layers, pooling, and image feature extraction.

### 📌 Week 04 – Variants in CNN Architecture

Advanced CNN architectures like LeNet, AlexNet, VGG, ResNet, and more.

### 📌 Week 05 – Explainable AI (XAI)

Model interpretability, SHAP, LIME, and understanding model decisions.

### 📌 Week 06 – CNN-Based Segmentation Algorithms

Image segmentation techniques including semantic and instance segmentation.

🔗 **Dataset (Road Signs - CNN Variants):**
https://drive.google.com/drive/folders/1d7F8hlnkHoRvALIfvKpNUDkxPE5F3DS7

---

### 📌 Week 07 – Sequence-to-Sequence Models

Encoder-decoder models, attention mechanisms, and applications.

### 📌 Week 08 – Long Short-Term Memory (LSTM)

Recurrent neural networks and LSTM architecture for sequence modeling.

### 📌 Week 09 – Natural Language Processing (NLP)

Text preprocessing, embeddings, and NLP pipelines.

### 📌 Week 10 – Autoencoders & Variational Autoencoders (VAE)

Dimensionality reduction and generative modeling.

### 📌 Week 11 – Large Language Models (LLMs)

Transformers, BERT, GPT, and modern NLP architectures.

### 📌 Week 12 – Advanced LLMs (Continuation)

Fine-tuning, prompt engineering, and applications of LLMs.
